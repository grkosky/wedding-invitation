# Wedding Invitation Customization Instructions

## Customizing the Wedding Invitation
Follow these instructions to customize the wedding invitation according to your preferences.

### 1. Couple Names
- Locate the section in the `index.html` or `invitation.html` file where the couple's names are displayed.
- Replace the placeholder text with the couple's actual names.

### 2. Date
- Find the date section near the top of the invitation files.
- Change the date to the actual date of the wedding.

### 3. Location
- Look for the location section in the invitation files.
- Update it with the venue name and address. Ensure it’s clear and easy to read.

### 4. Colors
- Open the CSS file (usually `styles.css`) to customize the colors.
- Search for color properties (e.g., `background-color`, `color`) and modify the values to your desired palette.

### 5. Fonts
- To change fonts, locate the `@font-face` or font import statements in the CSS file.
- You can add your preferred font and ensure it’s properly linked in the HTML files.

### 6. Audio File
- Find the section in the HTML file where the audio element is defined.
- Replace the `src` attribute of the audio tag with the path to your chosen audio file.

### 7. Configuring Links
#### RSVP Link
- Find the RSVP button or link in the HTML file.
- Update the `href` attribute with the URL of your RSVP form or webpage.

#### Google Maps Link
- Locate the map link section in the HTML file.
- Ensure it contains the correct Google Maps URL for the wedding venue.

#### Google Calendar Link
- Find the calendar link section in the invitation files.
- Update the link to allow guests to add the wedding date to their own calendars.

### Conclusion
Once you’ve made these changes, save the files. Your wedding invitation will be ready to send!
